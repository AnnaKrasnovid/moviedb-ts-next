## Инструкция по установке:
Клонируем репозиторий:
```
git clone https://github.com/AnnaKrasnovid/moviedb-ts-next.git
```
Переходим в папку проекта:
```
cd moviedb-ts-next
```
Устанавливаем зависимости:
```
npm install
```
Запускаем проект на компьютере:
```
npm run dev
```
